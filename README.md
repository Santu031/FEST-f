# Bhagat Singh Geleyar Balaga – FestFolio 🎉

A modern, responsive festival website built to showcase events, members, and sponsorship details of **Bhagat Singh Geleyar Balaga**, celebrating unity and tradition.

---

## 🚀 Features

- Dynamic event showcase with images and details  
- Members page highlighting team and contributors  
- Gallery inspired by social media layout  
- Sponsor section to display community partners  
- Fully responsive and mobile-friendly design  

---

## 🛠️ Tech Stack

- **React + TypeScript** – Component-based frontend  
- **Vite** – Lightning-fast development setup  
- **Tailwind CSS** – Modern utility-first styling  
- **shadcn/ui** – Elegant UI components  
- **Lucide Icons** – Clean, lightweight icons  

---

## ⚙️ Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <YOUR_REPO_URL>

# Navigate to the project folder

# Install dependencies
